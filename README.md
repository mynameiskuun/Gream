# Gream
의류 전문 온라인 쇼핑몰.<br><br>

### 프로젝트 목표

* 신입 개발자로서 갖춰야 할 기본적이지만 꼭 필요한 기능들을 직접 구현.
* MVC 모델, Layered-Architecture에 입각한 역할의 분리.
* 람다, 스트림, 3항 연산자를 적극 사용하여 코드의 가독성은 높이고, 복잡도를 최대한 낮추는 것에 집중.
* 이해를 기반으로 한 코드 작성

## 구현 기능

⭐️ 구매자<br>
* 로그인, 회원가입
* 상품 조회, 주문, 결제
* 쿠폰 저장, 사용
* 댓글 작성, 삭제
* 포인트 사용
* 장바구니 저장, 추가, 삭제
* 리뷰 작성, 문의글 작성.


⭐️ 관리자<br>
* 상품 조회, 등록, 삭제
* 판매기록 조회, 매출현황 조회
* 배송처리
* 쿠폰 발급, 삭제
* 주문내역 메일 발송.
