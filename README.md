# Gream
의류 전문 온라인 쇼핑몰.

### ⭐️ [프로젝트 바로가기](http://ec2-3-36-228-95.ap-northeast-2.compute.amazonaws.com)

### ⭐️ 프로젝트 기간
2023/04/15 ~ 진행중

### ⭐️ 프로젝트 목표

* MVC Model, Layered-Architecture에 입각한 역할의 분리.
* 람다, 스트림, 3항 연산자를 통한 코드의 가독성 향상.
* 이해를 기반으로 한 코드 작성.

### ⭐️ 프로젝트를 진행하며 경험하고 체득한 부분

* Spring Security(로그인)
